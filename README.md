# Adaptive Statistical Arbitrage in Commodity Markets

Analyzing how regime shifts and price spreads evolve across major commodities.

---

## Overview

This project explores adaptive statistical arbitrage strategies across global commodity markets such as crude oil, natural gas, gold, and copper.

It studies how price spreads behave under different volatility regimes and builds a backtestable system that adapts trading logic to market conditions.

## Tools and Libraries

- **Python**, **Pandas**, **NumPy**, **Statsmodels**

- **ARCH**, **hmmlearn**, **Pykalman**

- **Backtrader**, **Vectorbt**, **Pyfolio**

- **Matplotlib**, **Plotly**, **Seaborn**
